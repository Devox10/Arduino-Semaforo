# Arduino-Semaforo
Sketch Arduino che simula un semaforo stradale, utilizzando un Arduino Uno

# Funzionamento del circuito

Il circuito funziona come un normale semaforo stradale:
1. Il led rosso si accende per 5 secondi
2. Successivamente il led rosso si spegne e si accende il led verde per 5 secondi
3. Terminato il led verde si accende il led giallo per 2 secondi e si ricomincia da capo

# Materiale utilizzato

* Scheda Arduino
* Breadboard
* Cavi Jumper
* 3 Led
* 3 resistenze 220ohm
